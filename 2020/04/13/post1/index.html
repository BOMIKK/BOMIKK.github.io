Hexo를 이용하여 블로그 시작하기

마크 다운으로 적어보는 블로그 첫 글

hexo vs jekyll = node.js vs Ruby

몇년 전 동아리 활동을 통해 Ruby on ralis 를 배웠던 적이 있어 익숙한 것을 고르자면 루비지만
요새 관심을 두고 있는건 node.js와 React 쪽이라 hexo를 선택 했다.

검색해보면 hexo로 블로그 생성부터 테마 생성 등등 다양한 정보를 얻을 수 있었다.
처음 시작은 늘 그렇듯 install 부터다.

  1. npm install hexo-cli -g

  2. cd로 원하는 폴더로 이동 후

  3. hexo init (원하는 폴더명)

  4. server 실행 해보기! - > hexo server

중간에 났던 오류들

  1. _config.yml에 설정 중 오류

deploy:
type: git
repo: “깃 주소”
branch: master

ERROR Deployer not found: git
이번 오류는 단순하다. hexo 폴더에 깃 설치를 하지 않았다..^^;;

npm install hexo-deployer-git –save 로 해결!

  1. 테마 적용 오류
    사실 처음엔 정리되어 있는 테마를 설정하려다 새 테마들이 기본 적으로 가지고 있는 메뉴 들과 파일이 많아
    일단 기본 테마인 landscape에 어느 정도 적응을 해보려 한다. 전에 테마 적용 중 _config.yml에 theme에 제대로 입력을 했지만 .js .css가 로컬에서 적용되지않았다.

타 블로그 댓글에서 알려주신 방법을 통해 해결했다.
-> cd 로 적용 할 테마 폴더 까지 이동 후

서버 재 실행을 하면 이쁘게 적용이 잘 되어있다.
