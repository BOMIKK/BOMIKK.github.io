Hexo 블로그 꾸미기

2020-04-24

#Hexo #Blog

이전 글

Hexo를 이용하여 블로그 시작하기

1. 테마 고르기 및 적용

<anodyne 테마> 를 골랐다. 깔-끔

  • anodyne: https://github.com/klugjo/hexo-theme-anodyne

_config.yml에서 테마 폴더명으로 바꿔주면 끝!

2. 프로필 사진 바꾸기

anodyne 테마의 이미지 명은 /assets~로 되어있다. 이미지 경로에 추가하고

이런식으로 원하는 곳에 넣으면 끝!

3. 새로운 페이지 생성해서 꾸미기 (about, categories)


_posts 밑에 about,categories 폴더와 index.md가 추가되고 generate후엔


public 내 about,categories 폴더와 index.html가 추가 된다.


index.md안에 type: “이름” 를 추가하면 .md파일이 아닌 .html에 있는 내용이 적용된다.

4. github 카드 추가

https://www.npmjs.com/package/hexo-github-card

설치 먼저 한 후

.md 포스트에 추가 시 아래와 같이 적어주면 된다.

.html에는 아래와 같이 적어주면 된다.


post에 넣어서 했을땐 이런 식으로 들어가졌다.. about엔 div로 바꿈

5. 오류

  1. 꾸미고 generate후 deploy를 했는데 css등이 적용이 안되어있다.
    clean을 해봤지만 안되는 것을 보아 오류인거같은데 더 찾아보는 중..
    => clean을 하고 deploy 하고 시간이 지나니 잘 되어있다.

  2. 2번 프로필 사진 적용이 갑자기 안된다.
    => jpg 적용시 안되서 svg 파일로 변환해서 해봤지만 안보인다…
    => 경로를 못찾아서 적용이 되질 않았다. 포스트 이미지 폴더 내에 프로필 이미지 파일을 옮겨 실행하니 보인다.

  3. about 페이지 html 적용이 안된다.
    => hexo new page “about“ 이 아니라 hexo new page “About“

  4. generate만 하면 about페이지가 초기화된다.
